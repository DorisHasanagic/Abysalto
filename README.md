2. ### Getting Started
   #### Prerequisites
   - Java JDK (version 25):
        java -version
   - Maven (version 3.x)
        mvn -version 

   #### Installation
   - Clone the repository
   git https://github.com/DorisHasanagic/Abysalto.git
   
   - cd Abysalto 

   - Install Dependencies
   mvn clean install 

   #### Run the Tests
   - mvn test
   - from IDE: right click on class - run
